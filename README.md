# Full Stack Project

This is a full-stack web application inspired by Airbnb, where users can create, view, edit, and delete property listings. The project demonstrates complete backend architecture, authentication, and image upload functionality.

---

##  Features

-  User Authentication (Login / Signup using Passport.js)
-  Create, Edit, Delete Listings
-  Image Upload using Cloudinary
-  Server-side Validation using Joi
-  MVC Architecture (Model-View-Controller)
-  Full CRUD Operations
-  Dynamic UI using EJS Templates

---

##  Tech Stack

###  Backend
- Node.js
- Express.js

###  Database
- MongoDB (Mongoose)

###  Frontend
- EJS
- CSS

###  Authentication
- Passport.js (Local Strategy)

###  Other Tools
- Cloudinary (Image Upload)
- Joi (Validation)
